# Android Security Lab

## Overview

This repository documents my Android security lab built for learning ethical mobile security concepts in a controlled virtual environment.

The objective was to understand Android application architecture, emulator-based testing, network communication, and common security assessment workflows using virtual machines.

---

## Lab Environment

- VMware Workstation
- Kali Linux
- LDPlayer Android Emulator

---

## Topics Explored

- Android emulator setup
- Virtual lab configuration
- Android application structure
- Android security testing concepts
- Network connectivity between Kali Linux and Android emulator
- Meterpreter session analysis (controlled lab only)
- Android Remote Administration Tool (RAT) framework analysis
- Mobile malware research
- Android permissions and application behavior
- Security testing workflow

---

## Skills Practiced

- Linux
- Virtualization
- Android Security
- Mobile Penetration Testing
- Network Configuration
- Metasploit Framework
- Ethical Hacking Lab Setup
- Cybersecurity Research
